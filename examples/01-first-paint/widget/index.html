Splash Screen

Demo Application

This is a Video Splash Screen demo for first-paint splash configuration. This screen should appear when the page is read and the DOM structure is known. If there are still some out-of-line elements, like images or async script, they will be loaded after this step.

Demo calls console.log at certain points. Visible through DevTools, those logs can help see the relationship between splash screen and HTML application.
